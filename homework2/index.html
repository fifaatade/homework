The Basic Language of The Web: HTML


Posted by Laura Jones on Monday, June 21st 2027

All modern websites and web applications are built using three fundamental technologies : HTML, CSS and JavaScript. These are the languages of web.

In this post, let's focus on HTML. We will learn what HTML is all about, and why you too should learn it.

What is HTML ?

HTML stands for HyperText Markup Language. It is a markup language that developers use to structure and describe the content of a wepage (not a programming language).

HTML consits of elements that describe differnt types of content: paragraphs, links, heading, image, video, etc.. Web browser undersatnd HTML and render HTML code as websites.

In HTML, each elemnt is made up of 3 parts:

  1. The opening tag
  2. The closing tag
  3. The actual element
